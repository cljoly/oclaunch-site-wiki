You can use some variable to change the program behavior: Variable Usage ------------- ------------------------------------------------------------------- `OC_TMP` Change temporal file path `OC_RC` Change rc (i.e. configuration) file path `OC_RC_OLD` Chang old rc file, usefull to import from another file `OC_NOCOLOR` Disable color usage, if set to anything different of 0 `OC_YES` Always answer yes to question, if set to 1 `OC_VERB` Set verbosity, for instance 5 is very verbose whereas 0 is quiet
